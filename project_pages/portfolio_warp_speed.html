I developed Warp Speed as part of the UCSC GDA microgame jam. The theme of the jam was Speed or Size. After the jam was complete all microgames were put together to create a minigame collection. As part of this project I utilized Unity and C# to make the game fit alongside pre-existing features of the minigame such as the timer. The largest issue I had was fighting to the jamâ€™s theme. I eventually created rings that would randomly spawn at different heights at the side and would speed up the character. I created all the art for the game in GIMP and skinned the game as a small ship flying through rings to speed up. This theme of speeding up ensured that my minigame would mesh well with others through the theme of Speed or Size.
